# Projets et questions

Un **projet** est le conteneur principal de votre étude. Il regroupe les questions ouvertes, les fichiers de données et les sessions d'analyse associées.

## Créer un projet

Depuis la page d'accueil, cliquez sur **+ Nouveau projet**. Renseignez :
- **Nom du projet** — visible uniquement par vous (ex. : `Baromètre marque 2026 – vague 1`)
- **Description** *(optionnel)* — pour vous rappeler du contexte

## Ajouter des questions

Dans un projet, chaque question ouverte de votre questionnaire correspond à une **question VeraVoxa**. Cliquez sur **+ Ajouter une question** et saisissez :
- Le **libellé** exact de la question (ex. : `Pourquoi avez-vous choisi cette marque ?`)
- Optionnellement, un **plan de codes** ou un **plan de marques** à utiliser par défaut pour cette question

:::tip Plusieurs questions par projet
Un projet peut contenir autant de questions que nécessaire. Si votre questionnaire comporte 5 questions ouvertes, créez 5 questions dans le projet.
:::

## Importer les fichiers de données

Voir [Importer vos données](/importer-vos-donnees).

## Organiser ses projets

Vos projets apparaissent dans la liste principale, triés par date de dernière modification. Vous pouvez :
- **Renommer** un projet à tout moment
- **Archiver** un projet terminé pour garder votre liste épurée
- **Supprimer** un projet (supprime également tous ses fichiers et sessions — irréversible)

## Exemple : projet enquête satisfaction

Imaginons une enquête avec 3 questions ouvertes :
- Q1 : *Qu'est-ce qui vous a le plus satisfait ?*
- Q2 : *Quels points d'amélioration suggérez-vous ?*
- Q3 : *Quelles marques associez-vous à la qualité ?*

Vous créez un projet `Enquête satisfaction juin 2026`, puis 3 questions. Pour Q3, vous associez un plan de marques afin que VeraVoxa normalise automatiquement les noms cités.
