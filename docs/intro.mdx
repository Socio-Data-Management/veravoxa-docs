# Bienvenue sur VeraVoxa

**VeraVoxa** est une plateforme en ligne qui vous permet d'analyser automatiquement les réponses ouvertes de vos enquêtes et questionnaires. En quelques minutes, transformez des centaines ou des milliers de verbatims bruts en données structurées et exploitables.

## Ce que vous pouvez faire avec VeraVoxa

| Besoin | Service VeraVoxa |
|---|---|
| Regrouper les verbatims par thèmes | **Codification** |
| Détecter les émotions (positif / négatif / neutre) | **Analyse de sentiment** |
| Normaliser les noms de marques cités | **Recodification de marques** |
| Traduire les réponses dans une autre langue | **Traduction** |
| Supprimer les données personnelles | **Anonymisation** |
| Corriger les fautes de frappe | **Correction orthographique** |

Vous pouvez combiner plusieurs services sur le même fichier en une seule session.

## Comment ça marche ?

1. **Créez un projet** et ajoutez vos questions ouvertes.
2. **Importez votre fichier Excel** contenant les réponses.
3. **Lancez une session d'analyse** en choisissant les services souhaités.
4. **Téléchargez le fichier enrichi** : votre Excel original avec les nouvelles colonnes ajoutées.
