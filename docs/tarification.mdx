# Tarification

VeraVoxa facture à l'usage : vous payez uniquement les verbatims effectivement traités. Pas d'abonnement, pas de frais fixes.

## Principe de calcul

Le tarif d'une session dépend du **nombre de verbatims** et du **service** appliqué. Chaque service est tarifé séparément. Le total d'une session est la somme des tarifs de chaque service.

:::info Transparence totale
Le tarif exact est affiché avant tout paiement. Vous validez le montant avant de passer à la caisse — aucune surprise.
:::

## Tarifs par service

| Service | Base de calcul |
|---|---|
| Codification | Nombre de verbatims × tarif unitaire progressif |
| Analyse de sentiment | Nombre de verbatims × tarif unitaire |
| Recodification de marques | Nombre de verbatims × tarif unitaire |
| Traduction | Nombre de verbatims × tarif unitaire |
| Anonymisation | Nombre de verbatims × tarif unitaire |
| Correction orthographique | Nombre de verbatims × tarif unitaire |

Le tarif unitaire de la codification est **dégressif** : plus votre plan de codes est grand, plus le tarif par verbatim est ajusté en conséquence.

## Types de comptes

| Type | Mode de paiement | Tarif |
|---|---|---|
| **Visiteur** | Paiement immédiat par carte | Tarif de base |
| **Compte commercial** | Facturation différée (fin de mois) | Tarif de base + 10 % |
| **Compte staff SOCIO-DM** | Facturation interne | Tarif de base + 10 % |

Le surcoût de 10 % sur les comptes commerciaux et staff couvre les délais de paiement différé et la gestion administrative des bons de commande.

## Facturation et TVA

### Visiteurs (particuliers et entreprises françaises)
La TVA française en vigueur est automatiquement appliquée. Le montant TTC est affiché avant paiement.

### Entreprises hors France (Union Européenne)
Si vous renseignez un numéro de TVA intracommunautaire valide dans votre profil, la TVA n'est pas facturée (autoliquidation). VeraVoxa vérifie la validité du numéro en temps réel.

### Entreprises hors Union Européenne
Aucune TVA n'est facturée.

## Paiement

Les paiements sont sécurisés par **Stripe**. VeraVoxa accepte :
- Cartes Visa, Mastercard, American Express
- SEPA Debit (pour les comptes commerciaux sur facture)

## Factures

Toutes vos factures sont disponibles dans **Mon compte → Factures**. Chaque facture détaille :
- Le nombre de verbatims traités par service
- Le tarif unitaire appliqué
- Le montant HT et TTC
- La TVA (si applicable)

Vous pouvez télécharger vos factures en PDF à tout moment.

## Questions fréquentes sur la facturation

**Que se passe-t-il si je modifie une session déjà payée ?**
Si le nouveau périmètre est plus coûteux, vous payez uniquement la différence. Si le périmètre est réduit, aucun remboursement n'est effectué — la différence reste créditée sur votre compte.

**Puis-je tester avant de payer ?**
VeraVoxa propose un mode d'évaluation sur un échantillon de 10 % de vos verbatims (avec un minimum de 10 répondants). Le tarif de cet échantillon est inclus dans le tarif total affiché avant paiement.

**Comment obtenir une facture pour ma comptabilité ?**
Les factures sont générées automatiquement après chaque paiement et téléchargeables depuis votre espace **Mon compte → Factures**. Pour un compte commercial, les factures sont émises en fin de période selon votre accord commercial.
