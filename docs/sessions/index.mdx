# Qu'est-ce qu'une session ?

Une **session** est une demande d'analyse. Elle relie un ou plusieurs fichiers à un ou plusieurs services d'analyse, et produit en retour votre fichier enrichi de nouvelles colonnes.

## Anatomie d'une session

| Élément | Description |
|---|---|
| **Fichier source** | Le fichier Excel que vous avez importé |
| **Question(s)** | La ou les colonnes de verbatims à analyser |
| **Service(s)** | Le ou les traitements à appliquer (codification, traduction…) |
| **Résultats** | Le fichier enrichi, téléchargeable une fois l'analyse terminée |

## États d'une session

| Statut | Signification |
|---|---|
| **En attente** | Session créée, en attente de paiement |
| **En attente d'approbation** | Pour les comptes commerciaux — validation en cours |
| **En cours d'évaluation** | Un échantillon est analysé pour calibrer le tarif |
| **En cours d'analyse** | Traitement complet en cours |
| **Terminée** | Résultats disponibles au téléchargement |
| **Échouée** | Une erreur s'est produite — contactez le support |
| **Annulée** | Session annulée avant exécution |

## Combiner plusieurs services

Vous pouvez appliquer plusieurs services dans une même session. Par exemple :
1. **Correction orthographique** — VeraVoxa corrige les fautes avant l'analyse
2. **Codification** — sur les verbatims corrigés

VeraVoxa enchaîne automatiquement les services dans le bon ordre.

## Modifier une session

Une session peut être modifiée tant qu'elle n'est pas démarrée (statut *En attente* ou *Échouée*). Vous pouvez changer les services, ajouter une question, ou modifier les paramètres. Si le nouveau tarif est supérieur à ce qui a déjà été réglé, un complément de paiement vous est demandé.

:::info Pas de remboursement automatique
Si vous réduisez le périmètre d'une session déjà payée, la différence n'est pas remboursée. Elle reste créditée sur votre compte pour une prochaine utilisation.
:::
